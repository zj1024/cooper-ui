# Cooper UI

[![CircleCI](https://circleci.com/gh/zj1024/cooper-ui.svg?style=svg&circle-token=edd21a4ba4a00219f00b09f122b44eb3b52e1c28)](https://circleci.com/gh/zj1024/cooper-ui)

# Start

```
cnpm install
```

```
npm start
```

# Build

Library build

```
npm run publish:pre
```

Docs build

```
npm run build:docs
```

# Test

```
npm test
```
